# Sentiment


Web3 Dapp which allows u to vote if the token goes up or down. Vote ur sentiment and let the world know how u feel abt the token.

Tools used:

*Moralis(backend)
*react(frontend)
*solidity(smartcontracts)
*Polygonscan


network: mumbai-test-faucet
